# Watch your savings

The status-bar item shows the estimated tokens saved in your window. Open the
report to see two-tone bars — the **full bar is the total original tokens**, the
**bright part is what compressor saved** — plus a breakdown by day, tool, and
mode. Hover any bar for the exact character breakdown.

Below the charts, an **actual usage** section shows authoritative token counts
from this project's Claude Code session transcripts — real usage, *not savings*
and not billable dollars.

Token figures are estimates (chars / 3.5); character counts are exact. There
are no percentage claims — measured savings come from `compressor benchmark`.
